Page not defined:

Your page was mentioned when building Api structure, however you haven't defined it's look. There are several ways to do that. For detailed explanation consult documentation of AModules3. Here are brief explanation:

If you need a static page, you can create template inside "templates" subdirectory. You can use <?$page?> or other tags which will be filled with $_GET data. Only part enclosed into "content" tag will be used from your file.

You may add a custom class for your new template if you want a more complex generation. To do that, create file page/.php and define class page_.

Finally you may customize your own templates, but you should set $this->template while initializing your page class.

Consult documentation to understand basics about api/page structure.
