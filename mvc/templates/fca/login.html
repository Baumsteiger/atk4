AModules3
AModules3 is a PHP5 framework which uses object oriented development approach to simplify web development. By taking away all the technical issues, developer can address business logic and instead of thinking how to parse POST data he now can spend his time thinking how to arange elements on the page. This framework is Open Software and is licensed under LGPL. That means - you can develop application like this with a minimum effort - see http://adevel.com/amodules3 for more information.
